# Security Policy

## Supported versions

Security fixes are applied to the latest release on [PyPI](https://pypi.org/project/bnnr/) and the `main` branch.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead:

1. Open a private report via [GitHub Security Advisories](https://github.com/bnnr-team/bnnr/security/advisories/new) if you have access, or
2. Contact maintainers through a private channel listed on [bnnr.dev](https://bnnr.dev/team).

Include:

- Description of the issue and impact
- Steps to reproduce
- Affected versions
- Suggested fix (if any)

We aim to acknowledge reports within 7 days.
